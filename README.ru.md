# WUI (Window User Interface Library)

WUI — это кроссплатформенная библиотека для создания графического интерфейса пользователя на C++. Библиотека использует C++17 и имеет минималистичный API. В настоящее время поддерживается Windows (XP — 11) и Linux (glibc 2.23 и выше).

WUI разработана с целью предоставления разработчикам простого и эффективного инструмента для создания кроссплатформенных GUI-приложений на C++. Библиотека позволяет сосредоточиться на логике приложения, а не на особенностях реализации интерфейса для каждой платформы.

## Особенности

- Кроссплатформенность: поддержка Windows и Linux, в планах — macOS
- Минималистичный API, простота использования
- Общие интерфейсы для рисования и обработки событий, независимые от платформы
- Поддержка тем оформления и локализации
- Возможность создания и интеграции пользовательских контролов
- Управление конфигурацией приложения (реестр Windows, ini-файлы, JSON)

## Архитектура

Проект основан на трех ключевых сущностях:

- **Window** — окно, которое принимает системные события и управляет контролами
- **Control** — визуальный элемент интерфейса (кнопки, поля ввода и т.д.)
- **Graphic** — интерфейс для отрисовки, абстрагирующий платформо-зависимые методы рисования
