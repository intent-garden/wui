# libWUI

WUI (Window User Interface Library) — это кроссплатформенная библиотека для создания графического интерфейса пользователя на C++. Библиотека использует C++17 и имеет минималистичный API. В настоящее время поддерживается Windows (XP — 11) и Linux (glibc 2.23 и выше).
